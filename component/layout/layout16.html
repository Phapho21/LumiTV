Definition
Use Layout 16 to allow users to browse broadcast programmingÂ or scheduling information for current and upcoming programming.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Display the title that represents the current screen. In general, the title can be the name of an application, a device, and etc.
  • 2. Display the current date and time.
  • 3. Display the brief info of the item currently focused, such as the synopsis, the representative image, etc..
  • 4. Display the preview of the item currently focused.
  • 5. Display the list of channels.
  • 6. Text items used in this area should include a main text.
    In general, a text item for a channel includes the number, name, and logo of each channel.
  • 7. If there are too many channels to browse, the filter can be provided.
  • 8. Display the shows according to channel and time. Generally, equally spaced time intervals are displayed in the first row.
  • 9. Text items used in this area should include a main text.
    In general, they include the title and broadcasting time of the show.
