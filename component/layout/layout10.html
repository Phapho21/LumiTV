Definition
Use Layout 10 to play videos or music.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Buttons to control playback should be provided. Buttons in this area should be composed of an icon and a tooltip.
  • 2. Playback slider used in Layout 10 should include a bar, a main text, a sub text and an icon to present playback status.
    The total duration would be a main text and the current playback time would be a sub text.
  • 3. Display the title of the content currently played.
  • 4. Buttons in this area should be composed of an icon and a tooltip. If a function will go into effect immediately, it can be provided on this area.
  • 5. Display the list of items related to the current content.
  • 6. Text items used in Layout 10 should include a main text.
  • 7. When the focus moves up and down in the list, display the vertical scroll bar on the side of the screen.
Initial State
  • - The focus is on the PLAY/PAUSE button.
