Principles

Design the layouts based on the following principles.
Principle 1

When using two or more components on the same screen, ensure that the scrolling direction within a component and navigation direction between components must not be the same to allow users to easily access each component.

Principle 2

Place components (such as buttons) frequently used on every screen to perform general actions, on the top right area of the screen.
Examples are the Search button, the Sign In button, the Options button, and etc.

Principle 3

Place components (such as buttons) used for allowing users to control content items, in the body area of the screen.
Examples are the Add button, the Delete button, and etc.

Principle 4

Place components used for allowing users to filter, sort or select content items, near the target list or grid in the screen.
Examples are the Sort button, the Filter button and the Select All button, and etc.
