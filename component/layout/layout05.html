Definition
TBD
Use Layout 05 to make people view found results.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. There is an input field to enter the search keyword.
  • 2. Display the search results in a 1-row grid.
  • 3. Image items used in this area should include a main image.
  • 4. When the focus moves from side to side in the grid, display the horizontal scroll bar below the grid.
  • 5. Display the title of the searched item currently focused or selected.
  • 6. Display the subtitle of brief info of the searched item currently focused or selected.
  • 7. Display the description of the searched item currently focused or selected.
  • 8. If the description is too long to be displayed its own area, a button to see the whole description should be provided.
  • 9. Each group of related items can have a title.
  • 10. Display the list of items related to the searched item currently focused or selected.
  • 11. Image items used in this area should include a main image.
  • 12. When the focus moves from side to side in the grid, display the horizontal scroll bar below the grid.
Initial State
  • - The focus is on the first(leftmost) item of the first(top) grid.
