Definition
Use Layout 14 to show the list of contents and may ask the user to select items and perform an action to selected items.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Display the title that represents the current screen. In general, the title can be the name of an application, a device, and etc.
  • 2. Display a list of items.
  • 3. Text items used in this layout should include a main text.
  • 4. Each group of related items can have a title.
  • 5. When the focus moves up and down in the list, display the vertical scroll bar on the side of the screen.
  • 6. Buttons to sort or to filter the items in a grid are located just above the grid.
  • 7. Buttons that will affect to a grid immediately are located just above the grid, such as Select All.
  • 8. Display the preview of the focused item.
  • 9. Buttons that confirm action to the items are located beside the grid.
Initial State
  • - The focus is on the first(top) item of the list.
