Layout 02

  • Definition
  • Specs
  • Examples
Definition
Use Layout 02 to display groups of contents (Such as different genres of movies) in rows.
Each group of contents is displayed directly underneath the previous group.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Display the title that represents the current screen. In general, the title can be the name of an application, a device, and etc.
  • 2. Only 1-row grids are permitted in Layout 02.
  • 3. Image items used in Layout 02 should include a main image.
  • 4. When the focus moves from side to side, display the horizontal scroll bar below the grid.
  • 5. Subheaders could be omitted. In that case, grids should be visually distinguished.
  • 6. When the focus moves across rows, display the vertical scroll bar on the side of the screen.
  • 7. Buttons for functions that will affect to the whole screen can be located on this area.
    They should be composed of only a text.
Initial State
  • - The focus is on the first(leftmost) item of the first(top) grid.
