Definition
Use Layout 07 to promote contents.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Display the representative image of the content.
  • 2. Display the title of the content.
  • 3. Display the brief info of the content.
  • 4. Buttons in this area should be composed of only a text.
  • 5. Display the description of the content.
  • 6. If the information is too long to be displayed, a Button to show whole information should be provided.
  • 7. Display the list of items related to the content.
  • 8. Image items used in Layout 07 should include a main image.
  • 9. When the focus moves from side to side in the grid, display the horizontal scroll bar below the grid.
  • 10. Each group of related items can have a title.
Initial State
  • - The focus is on the first(top-left) button.
