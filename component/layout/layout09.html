Definition
Use Layout 09 to view photos.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Buttons to move to the previous or next photo and to control slideshow should be provided. Buttons in this area should be composed of an icon and a tooltip.
  • 2. Display the date the photo was taken.
  • 3. Buttons in this area should be composed of an icon and a tooltip. If a function will go into effect immediately, it can be provided on this area.
  • 4. Display the list of photos.
  • 5. Image items used in Layout 09 should include a main image only.
  • 6. When the focus moves from side to side in the grid, display the horizontal scroll bar below the grid.
Initial State
  • - The focus is on the PLAY/PAUSE button.
