Definition
Use Layout 06 to display a on-screen keyboard and text input fields.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1.Display the title that represents the current screen. In general, the title can be the name of an application, a device, and etc.
  • 2. Input Field used in Layout 06 should include Text and Input Field. Input Field should not be covered with IME.
  • 3. Following IME specifications.
Initial State
  • - The focus is on the first(top) input field.
