Definition
Use Layout 01 to display groupings of related control or setting items.
View the list of groupings on the left, and focus on one to see its items on the right.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Display the title of the currently selected tab. If there is no tab, display the title of the list.
  • 2. Display a list of setting items.
  • 3. Text items in this layout should include a main text.
  • 4. Tabs enable content organization at a high level. Tab labels should succinctly describe the content within.
  • 5. Display the image that describes the item currently focused.
  • 6. Display the title of the item currently focused.
  • 7. Display the subtitle of the item currently focused.
  • 8. Display the description of the item currently focused.
Initial State
  • - The first tab is selected.
  • - The focus is on the first(top) item of the grid in the first tab.
