C_INFOTILE
Definition
Info tile provides rich information about current screen or focused item.
Usage
Image only
Image + Main/Sub/Body text
Interaction
Specs
Properties
Info tile consists of the following elements:
1. Image (Optional)
2. Main text (Optional)
3. Sub text (Optional)
4. Body text (Optional)
Info tile should contain at least one of properties.
