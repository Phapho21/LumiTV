C_CONTENTTILE
Definition
Content tile is consisted of image and text to inform what content is.
Usage
  • - Image should be representative what content is like logo or poster.
  • - Title is applied the content name.
  • - It is hard to distinguish the content as image, sub image can be applied on the text area. (ex. image=screenshot, sub image=logo)
Tile on the list
Content tile 2
Interaction
Specs
Properties
Content tile2 consists of the following elements:
1. Image (Optional)
2. Icon (Optional)
3. Main text (Optional)
4. Sub text (Optional)
5. Sub image (Optional)
6. Progress bar (Optional)
  • - if it is live content, display progress bar to inform time data.
