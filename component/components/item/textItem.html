C_TEXTITEM
Definition

Text item is text format item that selectable and one of list element.

Usage

A text item’s component can be different but the component’s position should be consistent.

Interaction
  • - Entering the item changes screen while the relevant item is performed.
  • - If enter the item with value, display setting component like dropdown list, switch, slider and spin control without screen changes.
Specs
Properties
Text item consists of the following elements :
1. Main text (Mandatory)
2. Sub text (Optional)
3. Main icon / Image (Optional)
4. Sub icon (Optional)
5. Switch (Optional)
  • - Recommend to use switch when it is possible to set the item’s value as on/off.
