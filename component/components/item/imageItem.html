C_IMAGEITEM
Definition
Image item delivers information as image format and can be independently used with the additional information.
Usage
  • - An image item must contain a representative image. If it needs to display more information, it is able to use sub image.
  • - You can display up to several icons on the thumbnail according to the condition.
  • - Display a progress bar to represent the progress information like the live broadcasting content.
  • - The image item type with a pagination component, entered item will be operated as toggle function.
Interaction
  • - Entering the item changes screen while the relevant item is performed.
  • - If enter the item having toggle function, item will be changed without screen changes. (**Refer to Toggle Button for details.)
Specs
Properties
Image item consists of the following elements :
1. Main image (Mandatory)
  • - A thumbnail must contain a representative image.
2. Sub image (Optional)
  • - When the information transfer is insufficient with the main image, it is available to add the sub image.
3. Icon (Optional)
  • - You can display up to several icons on the thumbnail.
  • - Recommend to use icons’ count minimized.
4. Main text (Optional)
  • - Minimize the information so that users can only read the essentials.
5. Sub text (Optional)
6. Progress bar (Optional)
  • - Use a progress bar to display a progress status. In this case, place it at the bottom of the representative image.
7. Pagination (Optional)
  • - If the item has toggle function, use the pagination to represent the number of functions.
