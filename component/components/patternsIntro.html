Overview

PATTERNS

Patterns are that formalized collection of components is used repeatedly.

COMMON PATTERNS

Common patterns are the group of patterns that are used in common on layouts.
It is supported as winset to be used in same format.

APP PATTERNS

App patterns are the group of patterns that are specialized in certain UI layouts.

The property table is defined to guide patterns as below. (**Refer to Component’s property table description for details)
