C_TIMEPICKER
Definition
Use time picker to set a time with changing values continuously.
Usage
  • - Display main text of setting value on the same line. (**Refer to TEXT ITEM for details)
  • - It is available to display as main text like; Start time, End time and so on.
Interaction
Interaction is same as Date Picker. (**Refer to DATE PICKER for details)
Specs
Properties

Time picker consists of Hour, Minute and AM/PM.
The order of Hour, Minute and AM/PM follows the standard date format used in each respective country.
Use the colon (:) between spin controls to distinguish hour and minute.

Time picker consists of the following elements:
1. AM/PM spin control (Mandatory)
2. Hour spin control (Mandatory)
3. Minute spin control (Mandatory)
4. Colon (Mandatory)
