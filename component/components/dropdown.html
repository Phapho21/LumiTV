C_DROPDOWN
Definition
A dropdown allows users to choose an item from a list of mutually exclusive values.
Usage
  • - The list of items in a dropdown should be similar or be related in some way.
  • - Dropdown is composed of the selectable button (or item) and dropdown list appearing on the screen.
  • - Display the open/close indicator to let users know that there are more options available and whether the dropdown is displayed or closed.
Interaction
Specs
Properties
Dropdown consists of the following elements :
1. Icon (Optional)
If the list of items has each icon, display the icon on the same position.
2. Button text (Mandatory)
Display the currently selected value in the top button.
3. Indicator (Mandatory)
Display the open/close indicator to let users know that there are more options available and whether the dropdown is displayed or closed.
4. Dropdown list (Mandatory)
5. Selection icon (Mandatory)
Displayed beside the current value.
