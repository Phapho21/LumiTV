C_LISTBASIC
Definition
Lists present multiple line items vertically as a single continuous element. (1xM)
Usage
1. Text Item Size.
  • - The text item size is determined by the components and composition.
  • - If text item sizes are different, make all text item sizes the same with the size of the biggest text item.
  • - You can separately define the size and components when a text item is focused on.
2. Indicator
  • - Display a scroll indicator when scrolling is available.
  • - A scroll indicator is the graphic itself that displays part of the next or previous item lying in the scrolling direction.
3. Group Title Bar
  • - Use a group title bar when the items on the list can be categorized so that users can easily access them. Display a group title bar for each group.

Interaction
  • - Looping is not supported.
  • - If scroll starts, scroll bar will be displayed and same with scroll direction.
  • - The part of lists indicates that there is more lists over the screen.
Specs
Properties
Basic lists consist of the following elements :
1. Item Type (Mandatory)
  • - use the text item (**Refer to Text Item for details)
2. Selection icon (Optional, Mandatory for selecting from multiple items )
  • - Provided edit function as displaying checkbox or check mark(v)
