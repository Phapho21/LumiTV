C_LISTOPTION
Definition
Option list is a small, popuped list to change the setting value or provide entry point, option list provides related list with no screen changes.
Usage
Interaction
  • - Looping is not supported.
  • - If scroll starts, scroll bar will be displayed and same with scroll direction.
  • - The part of lists indicates that there is more lists over the screen.
Specs
Properties
Option list consists of the following elements :
1. Icon (Optional)
  • - Should be representative option list
2. Text (Mandatory)
