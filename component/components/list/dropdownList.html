C_LISTDROPDOWN
Definition
Dropdown list allows the user to choose one value and applies to entry point.
Usage
  • Dropdown button
  • Text item with dropdown lists
Interaction
  • - Looping is not supported.
  • - If scroll starts, scroll bar will be displayed and same with scroll direction.
  • - The part of lists indicates that there is more lists over the screen.
Specs
Properties
Dropdown lists consist of the following elements :
1. Icon (Optional)
  • - Should be representative option list
2. Text (Mandatory)
3. Selection icon (Mandatory)
  • - One of the option lists should be selected with checkmark(v).
