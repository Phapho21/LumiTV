C_RADIOBUTTON
Definition
Radio buttons allow the user to select one option from a set. Use radio buttons for exclusive selection.
Usage
  • - Radio buttons allow the user to select one option from a set. The set must has 2 or more options.
  • - Use radio buttons for exclusive selection if you think that the user needs to see all available options side-by-side. Otherwise, you can consider a dropdown to save space.
  • - To the multi-selection, it is recommended to use check box. (**Refer to C_CHECKBOX for details)
Interaction
Select / Deselect
Specs
Properties
A radio button consists of the following elements :
1. Circle icon (Mandatory)
  • - Provide a visible circle icon to let users know this control can be selected.
  • - Make a radio button dimmed when it is unavailable or disabled.
2. Check mark (Mandatory)
