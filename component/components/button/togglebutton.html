C_BUTTONTOGGLE
Definition

A toggle button allows the user to change a setting between 2 or more states.

Usage
  • - Use this button to allow users to instantly see the current status of an option and to easily change its status.
  • - Display the current status with an icon in the normal state, and display the button label in a tooltip when it is focused on.
  • - The button icon should be a visual cue that clearly shows the current status to the users.
Interaction
  • - Selecting the toggle button (with the ENTER button) will switch to the next value.
  • - Selecting the toggle button at the last value, it will return to the first value.
Specs
Properties
Toggle button consists of the following elements :
1. Icon (Mandatory)
  • - Represent the current status of an option.
  • - Change when the status changes.
2. Tooltip (Mandatory)
  • - Should be related on icon
  • - The text label alone should clearly describe the button action.
  • - Provide the text label in a tooltip.
