C_PREVIEWPANE
Definition
Preview pane is the window that keeps live TV experience as a part of screen.
Usage
Live TV
  • - Provide preview pane for Live TV source, information order should be like : CH number, CH name, Program title
  • - It is available to apply icon information optionally, applying information order should be like
    : Favorite, Lock, Scrambled (DVB only), Dynamic SI(DVB only)
Interaction
Specs
Properties
Preview pane consists of the following elements:
1. Preview area (Mandatory)
1. Main text (Optional)
1. Sub text (Optional)
1. Icon (Optional)
