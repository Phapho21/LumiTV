C_TAB
Definition

Tab makes it easy to explore and switch between different views or functional aspects of an app or to browe categorized data sets.

Usage

Each tab should contain the same type of content and must not contain different kinds of content in the same tab.

  • - Each tab should contain the same type of content and must not contain different kinds of content in the same tab.
  • - Use a category tab to present grouped content on one screen.
  • - Category tabs can be used in applications where users may want to quickly browse multiple types of content.
Interaction
  • - One of the tabs must always be selected.
  • - Looping is not supported.
  • - When a certain tab is focused, the corresponding content should appear below.
  • - When there are more tabs that are not currently shown on the screen, part of the next tab must be shown to let users know more tabs are available.
