C_POPUPBASIC
Definition
Use a message popup to display error messages, status information, or the result of users’ actions, or to ask for confirmation.
With this popup, users can also proceed to the step.
Usage
Specs
Properties
Message popup consists of the following elements :
1. Main text (Mandatory)
  • - Type the popup title
2. Sub text (Optional)
  • - Non-scrollable area
3. Custom area (Optional)
4. Button (Mandatory)
