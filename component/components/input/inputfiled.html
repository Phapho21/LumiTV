C_INPUTFIELD
Definition
Input Field is used when user input character or number manually.
Usage
  • - Use a text field when users need to enter characters or characters and numbers at the same time.
    Provide an input method editor (IME) when the text field is selected. (**Refer to IME Overall Interaction IA Guideline for IME specification.)
  • - Provide a label for a text field to aid user's comprehension.
  • - Use hint text when it’s necessary to provide users with more information about what they have to enter.
  • - Hint text can be a guide message or the latest value that a user previously typed into the text field.
Interaction
Specs
Properties
Input field consists of the following elements :
1. Input field (Mandatory)
  • - Provide an input field that allows users to enter text.
2. Text (Optional)
  • - Provide text field to aid user’s comprehension.
3. Hint Field (Optional)
  • - Use hint text when it’s necessary to provide users with more information about what they have to enter. If a hint text is not necessary, it can be left blank.
  • - Display the latest value that a user previously typed into the input field if it should be continuously shown to the user.
  • - Display asterisks (*) instead of the actual password when it is typed.
4. Cursor (Mandatory)
  • - A cursor should blink when users is entering text into the text field.
