Lock / Unlock

  • Definition
  • Usage
Definition

The Lock and Unlock functions are used for locking an item to deny access or for unlocking a locked item to allow access.

Usage

PIN confirmation should be performed when users execute Lock or Unlock functions or select a locked item.
When a valid PIN is entered, the user proceeds to the next step. If the entered PIN is invalid, the UI should require users to re-enter their PIN.

The title of a popup should be the name of the current action - Lock or Unlock.

The interaction and information about PIN follows PIN component.
(**Refer to 1-line PIN Field for details)
