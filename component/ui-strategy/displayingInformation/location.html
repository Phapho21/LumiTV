Location

  • Definition
  • Usage
Definition

Use the Location to give users information about the current location when a lower depth is accessed in the UI hierarchy.

Usage

Display the upper-level path and the current location.

In case it is difficult to display the total path with the space constraint, omit from the topmost path in order.
Replace the whole path to ellipsis (…) for the omitted path, and don’t display the part of the path.

In case it cannot display the whole current location even though omitting the upper path, place the ellipsis (…) at the end of the current path.
