Voice Contextual Command

  • Definition
  • Usage
Definition
  • - This feature is for executing functions or providing guide related to the currently focused item by saying the designated contextual commands.
  • - Not only with the information of current focus, the external context can be used for execution.
  • - Refer to the Voice Interaction IA Guideline / All Cases Document for more details about the command guide.
Usage
Voice Key (By Command) (TBD)
Link to E-manual (Setting UI)

When the focus is located at a certain setting item on the setting UI, if the user starts voice chrome and commands “E-manual”, the E-manual App will be launched and automatically jump to the relative page with the currently focused item. For checking the supporting items, refer to the E-manual specification.

  • - For UI controlling context (Eden UI, Search UI, …) (TBD)

Contextual command guide by time out
  • - If the user starts voice chrome but ends the recording without saying nothing or ends by timeout, the same key as the command for providing help will be sent.
    The guide will be provided relatively by the context. Only the valid commands are provided for useful commanding for next time.
