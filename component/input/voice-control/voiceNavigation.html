Definition
  • - This feature is for moving focus or selecting item to execute on the UI which is actually seen on the screen by saying voice navigation command.
  • - For moving focus or selecting Nth item, it is carried out on the item list where the current focus is located.
  • - For erroneous or incapable cases, relative error response message will be displayed on the voice chrome.

Specs
Properties
- Mandatory / Option
- Limitations
  • . Voice navigational command should be registered as a Voice Key in general.
  • . Some specific voice navigational command can be registered as a App-Control Command.
- Rules for the item inside / item outside
- Inter-Component Navigation (※ The rule when the List / Tab / Button is displayed in the same screen)
  • . Moving multiple steps is valid only within the same list of the content.
  • . For moving across the component, use the single step movement.
