Definition

Highlight acts as the user's eyes and it represents the item, in which the user's focus is currently on.
By using the remote control, it moves with the user's view and is able to be selected.

Properties
1. Clear & Precise

Tension effect is applied to give clear feedback and high visual hierarchy during the moment highlight takes place.

2. Visual Consistency

Unified style highlight effect should be applied to all elements.
