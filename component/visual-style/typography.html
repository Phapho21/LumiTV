Weights
When composing the screen with text, it is possible to create more aesthetically pleasing screens by using different text weights depending on thetype of information. Samsung One UI font comes in range of 6 different weights (200, 300, 400, 500, 600, 700) and 300, 400 and 600 are usedtogether in System UI to create a more varied rhythm.

Leading
Leading should be kept to 1.5x the font size. Apply defined leading when using size from the Font Size Set.
Color & Contrast
Color

All fonts are highly recommended to use the default color values. Under certain circumstances such as warning message and selected items, the defined colors down below can be applied.

Normal Color #555555
Focused Color #111111
Point Color #208bbd
Warning Color #f86900
Contrast

The legibility for the texts should be legible throughout.
In order to maintain best legibility, the luminence on background color should be kept under 20%.
Also, *contrast ratio between text and background color (contrast ratio) should be greater than 3:1.

*Following the International ISO standards
A contrast ratio of 3:1 is the minimum level recommended by [ISO-9241-3] and [ANSI-HFES-100-1988] for standard text and vision.
Text Sliding
Sliding

For texts that go over the designated length area within items, apply gradient masking for normal status.
For focused status, slide 50px per second after the item has been on focus for 1 second.

Text Align

For Title Text, Partial UI ones are applied with center alignment and left alignment for Full UI titles.
Rest of the text follows left alignment unless mentioned otherwise specifically.
For Popup, the Message Popup follows rule where single-text is with center alignment and for texts with 2 or more lines, apply the left alignment rule.

Title
  • Partial UI Title : Center Align
  • Full UI Title : Left Align
Contents
  • List : Center Align
  • List : Left Align
  • Message Popup text single line : Center Align
  • Message Popup text single line : Left Align
Icon

For cases where icon and text are used together, if text is placed on either left or right side of the icon, apply left alignment.
For texts placed either on top or bottom of the icon, apply center alignment rule.

  • Text is placed on either left or right side : Center Align
  • Texts placed either on top or bottom : Left Align
