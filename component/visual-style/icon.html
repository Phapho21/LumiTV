2) 2.5D
2-1. Specific Device (R_ICON_SPECIFIC)
When representing actual product or device, the device’s characteristics are somewhat realistically represented.
Compared to 2D icons, this type requires for more specific representation techniqures, hence should be size at minimum 80x80 to represent more clearly. Pay careful attention to make sure these still represent as icons. The perspective will either be front or top-down view depending on which view best represents the device, and should refrain from photo-realistic representation techniques.

3) Logo

There are two different types of icons for logos in Samsung TV : channel and 3rd party app logos. As logos are to represent that particular broadcasting station or service, these should not be modified. However, as changing the outer shape of the logo does not destroy the actual logo, a circular or rounded rectangular frame is allowed. Samsung TV uses a unified outer shape for these types of logos.

3-1. Circle Frame
For channel logo, it uses a circular frame.
3-2. Square Frame
For apps, company services, and 3rd party apps which are shown as the app itself. a rectangular frame is applied.
Launcher icons in home are 16:9, all the rest are square-shaped.
