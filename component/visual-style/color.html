Definition
Color is an essential element to the UI’s impression and it allows for information status to be shown clearly. 2017 Samsung TV color palette uses diverse color spectrum to create an authentic and aesthetic imagery.
Color Pick

Color pick is the software technique that apply colors to the UI by picking up the colors in the screen that overlaps with the UI. This allows to minimize the difference between what’s shown on screen and the UI, as well as maintaining the elegance when UI is shown on screen. Samsung color palette consists of 9 differtent color chips to provide the most natural viewing experiene, regardless of the color spectrum in the screen.
