Dim

  • Definition
  • Usage
Definition

Dim is used to heighten the UIâ€™s legibility for cases where there is an overlap between the UIs due to another UI that appears on top.
However, if the UI appears on top of Live TV screen, the dim effect does not take place.
For cases where another UI appears on top of full screen size UI and 2 pieces of UI overlaps on top of another, entire dim and partial dim are applied.
Moreover, partial UI features like Eden, Notification, and Pairing are also exempt from the dim rule.

Usage
1. POPUP ON TOP OF APP SCREEN

When popup appears on top of app screen, apply 50% dim to the entire app screen. (App screen size Black background Opacity 50%)

2. SUBLIST ON TOP OF LIST

When sublist appears on top of list, apply 70% opacity to the list text. (List text Opacity 70%)

3. THUMBNAIL

When other thumbnails besides the selected thumbnail is inactive, apply 50% opacity to the thumbnails. (Thumbnail Opacity 50%)

4. MULTIPLE POPUPS DURING LIVE TV

When multiple popups appear during Live TV mode, apply 40% dim to popup without the focus. (Black background to be equal size as the popup Opacity 40%)

5. MULTIPLE POPUP DURING APP SCREEN

When multiple popups appear on App screen, apply 50% dim to the entire app screen and also apply 40% dim to the popup without focus.
(Black background to be equal size as the app screen Opacity 50% / Black background to be equal size as the popup Opacity 40%)
