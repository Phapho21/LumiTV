Layout 04

  • Definition

  • Examples
Definition
Use Layout 04 to display groupings of related items, such as genres of movies or TV shows. View the list of groupings on the left, and focus on one to see its items on the right.
If the primary distinguishing content consists of text, use Layout 04.
Layout 04 is best suited to presenting a homogeneous data type or sets of data types, typically text, and is optimized for reading comprehension while differentiating either between similar data types, or qualities within a single data type.
Specs
Properties
  • Detail view
  • Simple view
Note
  • 1. Display the title that represents the current screen. In general, the title can be the name of an application, a device, and etc.
  • 2. Display a list of items.
  • 3. Text item used in Layout 04 should include a main text.
  • 4. Each group of related items can have a title.
  • 5. When the focus moves up and down, display the vertical scroll bar on the side of the screen.
  • 6. Tabs enable content organization at a high level. Tab labels should succinctly describe the content within.
  • 7. Buttons for functions that will affect to the currently selected tab can be located just below the current tab.
  • 8. Display the image that describes the item currently focused.
  • 9. Display the title of the item currently focused.
  • 10. Display the subtitle of the item currently focused.
  • 11. Display the description of the item currently focused.
  • 12. Tabs enable content organization at a high level. Tab labels should succinctly describe the content within.
  • 13. Buttons for functions that will affect to the whole screen can be located on this area. They should be composed of only a text.
Initial State
  • - The first tab is selected.
  • - The focus is on the first(top-left) item of the grid in the first(top) tab.
